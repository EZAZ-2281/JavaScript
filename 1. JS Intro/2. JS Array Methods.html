*** JavaScript Array delete() Warning ! Array elements can be deleted using the JavaScript operator delete. Using delete leaves undefined holes in the array. Use pop() or shift() instead.

*** Merging (Concatenating) Arrays The concat() method creates a new array by merging (concatenating) existing arrays: The concat() method does not change the existing arrays. It always returns a new array.

*** The concat() method can take any number of array arguments: Example (Merging Three Arrays) const arr1 = ["Cecilie", "Lone"]; const arr2 = ["Emil", "Tobias", "Linus"]; const arr3 = ["Robin", "Morgan"]; const myChildren = arr1.concat(arr2, arr3); The concat() method can also take strings as arguments: Example (Merging an Array with Values) const arr1 = ["Emil", "Tobias", "Linus"]; const myChildren = arr1.concat("Peter"); *** Splicing and Slicing Arrays The splice() method adds new items to an array.
