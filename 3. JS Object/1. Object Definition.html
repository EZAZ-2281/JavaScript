JavaScript Objects In JavaScript, objects are king. If you understand objects, you understand JavaScript. *** JavaScript Primitives A primitive value is a value that has no properties or methods. A primitive data type is data that has a primitive value. JavaScript defines 5 types of primitive data types: string number boolean null undefined Primitive values are immutable (they are hardcoded and therefore cannot be changed). *** Objects are Variables It is a common practice to declare objects with the const keyword. For readability, simplicity and execution speed, use the object literal method.(মানে আগের পদ্বতি গুলো) *** JavaScript Objects are Mutable Objects are mutable: They are addressed by reference, not by value. If person is an object, the following statement will not create a copy of person: const x = person; // Will not create a copy of person. The object x is not a copy of person. It is person. Both x and person are the same object. Any changes to x will also change person, because x and person are the same object.
